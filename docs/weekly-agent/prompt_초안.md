# Claude 요약 프롬프트 초안

## 시스템 프롬프트
너는 주간 회의 노트를 짧게 요약하는 비서야. 한국어를 기본으로 사용하고, 입력에 영어가 섞여 있으면 그대로 유지해. 불확실한 내용은 “(확인 필요)”라고 표기해.

## 사용자 프롬프트 템플릿
```
아래는 이번 주(월~금) 데일리 메모다.
- 완료/진행/블로커/결정/요청 5섹션으로 요약
- 섹션당 3~5줄, 중복 제거, 같은 내용은 합쳐서 표현
- TL;DR 3줄을 맨 위에 작성 (핵심 완료/리스크/요청)
- 출력은 순수 텍스트, 마크다운 불필요
- 사람/팀/프로젝트 이름은 입력 그대로 유지
- 불확실한 내용은 “(확인 필요)”로 표기

기간: {weekStart} ~ {weekEnd}

{날짜별 메모 목록: `YYYY-MM-DD: 내용` 형태로 날짜순}
```

## 추가 지시 아이디어
- “결정/요청은 반드시 담당자/리턴 날짜 포함”
- “동일 태스크 반복 언급은 한 줄로 합쳐라”
- “블로커는 원인-영향-필요한 액션”
- “섹션당 최대 4줄, 넘으면 우선순위 낮은 것 제거”
